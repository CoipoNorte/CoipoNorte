https://github.com/CoipoNorte/tarantula
TARANTULA es una aplicación de escritorio para Windows que permite visualizar, filtrar y cerrar puertos de red en tiempo real. Diseñada para desarrolladores que necesitan gestionar rápidamente los puertos ocupados por servidores locales, bases de datos y servicios de desarrollo.

https://github.com/CoipoNorte/zorro
ZORRO es una aplicación de escritorio para Windows que genera paletas de colores profesionales a través de algoritmos sobre el círculo cromático. Cada color generado incluye su rol específico (fondo, texto, botones, etc.) para que tanto tú como la IA sepan exactamente para qué se usará cada color en el proyecto.

https://github.com/CoipoNorte/ranita
Ranita es una app de escritorio para Windows que convierte cualquier imagen en un .ico profesional con todas las capas que Windows necesita. Arrastra tu PNG, genera el icono y listo. Sin webs, sin límites, sin internet.

https://github.com/CoipoNorte/pudu
Pudu es un generador de árboles ASCII que escanea la estructura de archivos de cualquier proyecto y la presenta con iconos emoji, filtros inteligentes y una interfaz moderna con tema oscuro. Perfecto para documentación, READMEs y compartir la organización de tu código.

https://github.com/CoipoNorte/picaflor
Picaflor es una herramienta de escritorio construida con Tauri, React y Bootstrap 5. Analiza proyectos de código, detecta tecnologías, genera resúmenes estructurados y los deja listos para pegar en un chat con IA. En lugar de pasar 60 archivos uno por uno, Picaflor genera un solo texto que cualquier modelo de lenguaje puede entender de inmediato.

https://github.com/CoipoNorte/huillin
Huillin es una app de escritorio para Windows que captura secciones de pantalla, ventanas o pantalla completa y extrae texto de las imágenes con OCR. Como Recortes de Windows, pero con superpoderes. Sin internet, sin límites, sin suscripciones.

https://github.com/CoipoNorte/condor-desktop
Condor Desktop es un IDE visual de automatización que lee instrucciones desde archivos .md generados por IA y las ejecuta automáticamente. Crea archivos, modifica código, ejecuta comandos en terminal y construye proyectos completos desde una interfaz inspirada en VS Code. La evolución de escritorio del sistema CONDOR.

https://github.com/CoipoNorte/colo
COLO es una aplicación de escritorio para Windows que permite cambiar iconos de carpetas, accesos directos, archivos ejecutables (.exe) y convertir imágenes a formato .ico. Nombrada en honor al Colocolo, un felino silvestre endémico de Chile.

https://github.com/CoipoNorte/churrin-timer
Churrin Timer es una aplicación de escritorio moderna para programar el apagado automático de tu PC. Construida con Tauri, React y Bootstrap Icons, ofrece una interfaz compacta con efecto glassmorphism, bandeja del sistema y control total desde un diseño minimalista.

https://github.com/CoipoNorte/choike
Choike es un launcher profesional diseñado para automatizar la creación de proyectos React en tiempo récord. Olvídate de escribir comandos repetitivos en la terminal; selecciona tu stack, elige la versión y lanza tu próxima gran idea.
